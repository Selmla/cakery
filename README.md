# Munamii Cakery Website

This is a simple school project website built using only HTML and CSS. The goal of the project was to learn the basics of web development by creating a small, static bakery-themed website.

## Project Overview

- **Project name:** Munamii Cakery
- **Purpose:** Learn how to structure webpages with HTML and style them using CSS.
- **Technologies used:**
  - HTML
  - CSS
  - Google Fonts

## Website Structure

The website includes the following pages:

- `index.html` — Home page with a welcome section, navigation bar, and product call-to-action.
- `about.html` — About page for the bakery (project includes additional content here).
- `products.html` — Product page showcasing cakes and treats.
- `contact.html` — Contact page with a way to reach the bakery.
- `styles.css` — Main stylesheet for page layout, typography, and visual styling.
- `images/` — Image assets used throughout the website.

## Features

- Responsive navigation bar with links to all pages
- Hero section with a bakery name and subtitle
- Decorative image and product call-to-action buttons
- Animated or visually styled cake belt section for branding
- Footer icon links including contact email

## Learning Outcomes

This project was created to practice:

- writing semantic HTML structure
- using external CSS for layout and styling
- embedding custom fonts from Google Fonts
- organizing a small website project into multiple pages
- building a consistent visual theme for a bakery site

## How to View

1. Open `index.html` in a web browser.
2. Use the navigation menu to browse `about.html`, `products.html`, and `contact.html`.

## Notes

This is a static website meant for learning and demonstration only. No back-end logic or JavaScript is included.
